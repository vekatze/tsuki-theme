** Installation
Please use [[https://github.com/radian-software/straight.el][straight.el]] or something like that.

#+begin_src emacs-lisp
;; with straight.el
(use-package tsuki-theme
  :straight
  (:host github :repo "vekatze/tsuki-theme")
  :init
  (load-theme 'tsuki t))
#+end_src
